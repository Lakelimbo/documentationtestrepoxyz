About UniMaker

An easy-to-use level design tool based on Super Mario Maker. Featuring over twenty built-in themes and support for customized effects, graphics and songs that can be packed with your courses.

Download#

Download Super Mario UniMaker version 1.S via GameJolt.

(placeholder)

If you want to download the source code, head over to GithubGitHub and click on "Clone" then "Download ZIP".

And if you use Git, you can easily clone by using the following command line:

$ git clone https://github.com/MarioSilvaGH/UniMaker

What's included#

The source package contains a .zip folder with Super Mario UniMaker 1.S, 1.1, 1.0, Super Mario 2D Universe (legacy) and GFX Cat Tool (obsolete).

note

None of the source codes above contain songs in the file system due to copyright issues but you can extract them from compiled releases, follow these steps:

  • Download the compiled executable;
  • go to the "UniMaker" subfolder (where the executable is located) and then copy all the .ogg files;
  • open your project in GameMaker Studio, go to the "Included Files" folder visible in the resource tree then import all previously copied files;
  • done!

Requirements#

UniMaker and the extras included are made with GameMaker Studio and GameMaker Language. If you don't own or don't know any of these, you might check out GameMaker Studio's official website.

Oh yes, this pack also comes with Super Mario 2D Universe, an old project preceding UniMaker. Check its own documentation too (small, but still).
